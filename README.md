# Roblox Advanced Movement System

A custom Roblox movement system focused on smoother and more responsive player movement.

## Features

- Sprint
- Crouch
- Slide
- Momentum-based movement
- Camera bob
- Camera tilt
- Landing impact

## Controls

- Shift = Sprint
- C = Crouch
- LeftControl = Slide

## Installation

Place the script inside:

StarterPlayer > StarterPlayerScripts

as a LocalScript.
